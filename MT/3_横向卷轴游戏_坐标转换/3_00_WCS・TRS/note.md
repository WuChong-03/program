 ## step1 :  理解 `WCS坐标系`与`SCS坐标系`

 | 名称        | 用途                | Y轴方向 | 原点位置        |
| --------- | ----------------- | ---- | ----------- |
| 世界坐标 | 游戏逻辑用的 | 往上是正 | 自定义点 |
| 屏幕坐标 | 图像绘制用的            | 往下是正 | 左上角(0,0)    |

- 使用y轴为正的自定义坐标系, 符合物理逻辑方便计算,  最后转化为屏幕坐标绘制即可.
